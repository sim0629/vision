This folder "~/bottle/" contains 2 sets of images (each containing 14 image) and 2 image masks.

     bottle[1-14].png - 14 images of the target object with varying light sources

     sphere[1-14].png - 14 images of a reference sphere with varying light sources


     bottlemask.png - mask for the target object images (white = object, black = background)

     spheremask.png - mask for the reference sphere images (white = object, black = background)
