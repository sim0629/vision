This folder "~/wavy/" contains 2 sets of images (each containing 14 image) and 2 image masks.

     wavy[1-14].png     - 14 images of the target object with varying light sources

     wclinder[1-14].png - 14 images of a reference cylinder with varying light sources


     wavymask.png      - mask for the target object images (white = object, black = background)

     wcylindermask.png - mask for the reference cylinder images (white = object, black = background)
